# Software-Architecture-virtual-Healthcare-Java-project-
The objective of this assignment is to create Virtual Healthcare System (VHS) that facilitates remote patient-provider interactions and healthcare management. The system goals include patient/provider registration, scheduling virtual appointments, electronic health record management, prescriptions, billing, and feedback collection. 

This project is a simple console-based Virtual Healthcare System built in Java.  
It includes patient registration, login, viewing providers, and booking appointments.

## Classes Included
- User.java  
- Patient.java  
- HealthProvider.java  
- Administrator.java  
- Appointment.java  
- Feedbacks.java  
- Records.java  
- Treatments.java  
- VirtualCareSystem.java  
- Main.java
